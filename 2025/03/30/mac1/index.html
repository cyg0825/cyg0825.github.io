解决软件安装提示损坏，关闭SIP

在终端输入csrutil status

看状态如果是enabled，就把电脑关机，长按指纹键，进入一个两个按钮一个mac一个选项，选择选项然后进入终端

在终端输入csrutil disable

然后打开电脑输入csrutil status看看是否已关闭，是disabled说明SIP已关闭，如果还是提示软件损坏那就是软件本身是坏的

软件安装完之后重启关机，再把SIP关掉，不然不安全

和关闭一个步骤，输入csrutil enable

重启电脑查看一下SIP是都关闭csrutil status

命令行

基础命令

  1. 文件与目录操作
    1. ls：列出当前目录内容 ls -al：显示隐藏文件及详细信息
    2. cd [目录]：切换目录（cd ~ 返回用户主目录）
    3. pwd：显示当前目录路径
    4. mkdir [目录名]：创建新目录
    5. touch [文件名]：创建空文件或更新文件时间戳
    6. cp [源文件] [目标路径]：复制文件 cp -r [目录] [目标路径]：复制目录
    7. mv [源文件] [目标路径]：移动文件或重命名
    8. rm [文件]：删除文件 rm -rf [目录]：强制删除目录及其内容（⚠️ 慎用）
  2. 文本处理
    1. cat [文件]：查看文件内容
    2. nano [文件] 或 vim [文件]：文本编辑器
    3. grep "关键词" [文件]：搜索文件中的关键词
    4. head -n 5 [文件]：显示文件前5行
    5. tail -n 5 [文件]：显示文件末尾5行 tail -f [文件]：实时追踪文件更新（如日志）
  3. 权限管理
    1. chmod [权限] [文件]：修改文件权限（如 chmod 755 script.sh）
    2. chown [用户]:[组] [文件]：修改文件所有者
  4. 进程管理
    1. ps aux：查看所有进程
    2. kill [PID]：终止进程 kill -9 [PID]：强制终止进程
    3. top：实时监控系统进程（类似任务管理器）
  5. 网络相关
    1. ping [域名/IP]：测试网络连通性
    2. curl [URL]：下载或访问网页内容
    3. ssh [用户]@[IP]：远程登录其他主机
    4. ifconfig：查看网络接口信息（类似 Windows 的 ipconfig）

进阶命令

  1. 包管理工具（需安装 Homebrew****）
    1. brew install [软件名]：安装软件
    2. brew update：更新 Homebrew
    3. brew upgrade：升级已安装的软件
  2. 查找与统计
    1. find [目录] -name "文件名"：搜索文件
    2. du -sh [目录]：统计目录大小
    3. df -h：查看磁盘剩余空间
  3. 系统信息
    1. uname -a：显示系统信息（如内核版本）
    2. system_profiler SPHardwareDataType：查看硬件信息
    3. sw_vers：查看 macOS 版本
  4. 快捷键与自动化
    1. !!：重复上一条命令
    2. command1 && command2：前一条命令成功后执行下一条
    3. crontab -e：编辑定时任务

系统管理

  1. 管理员权限
    1. sudo [命令]：以管理员身份运行命令（需输入密码）
  2. 系统服务
    1. launchctl load [plist文件]：启动服务 launchctl unload [plist文件]：停止服务
  3. 安装/卸载软件
    1. 通过 .dmg 或 .pkg 安装的软件，卸载时可能需要手动删除相关文件（通常位于 /Applications 或 ~/Library）。
  4. 系统更新
    1. softwareupdate -l：列出可用更新 softwareupdate -i -a：安装所有更新

实用技巧

  1. 快速打开文件/目录
    1. open [文件或目录]：用默认应用打开（如 open . 打开当前目录）
  2. 清空终端屏幕
    1. 输入 clear 或快捷键 Ctrl + L
  3. 历史命令
    1. 按 ↑/↓ 键浏览历史命令 history：查看所有历史命令
  4. 生成随机密码
    1. openssl rand -base64 12：生成12位随机密码
  5. 让 Mac 说话
    1. say "Hello World"：Mac 会朗读这句话（娱乐功能）
