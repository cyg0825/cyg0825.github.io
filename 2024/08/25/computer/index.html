1.初始电脑

初识电脑

电脑:计算机、用于处理信息与数据

分类:台式电脑、笔记本电脑、一体机电脑、平板电脑等

组成:硬件(主机、显示器等)、软件(操作系统、应用程序等)

台式电脑

组成:机箱、显示器、鼠标、键盘、其余配件(耳机、音响等)

显示器:用于画面的显示，使用连接线进行连接

机箱:用于承托电子配件(CPU、主板、显卡、硬盘、散热器等)

鼠标: 操控系统界面

USB接口:数据传输

键盘:向电脑传输数据

音频接口:用于传输音频

缺点:硬件设备过多

网络接口:连接互联网

显示器接口:VGA接口、HDMI接口、DP接口

笔记本电脑

硬件:显示器、键盘、鼠标(触摸板)

特点:方便携带、便捷

配置接口:USB接口、音频接口、网络接口、显示器外界接口

操作系统

分类:Windows系统、MAC(苹果)系统、Linux系统

Windows系统分类:Windows XP、Windows7、Windows 10等

鼠标基础使用

鼠标分类与构成

分类:有线鼠标、无线鼠标(USB鼠标、蓝牙鼠标)

构成:鼠标左键、鼠标滚轮、鼠标右键、侧键

呈现方式:箭头、I状光标、手指

鼠标操作

左键单击:选中某一个对象(文件、文件夹等)

左键双击:打开某一个对象左键

长按:拖动某一些对象

鼠标右键:唤出不同对象的菜单选项

滚轮键:实现翻页功能与让屏幕自动滚动

鼠标修改

[菜单]-[设置]-[蓝牙和其他设备调整鼠标大小、颜色等]

触摸板

单指:与鼠标左键功能相似，单击、双击等

双指:与鼠标右键功能相似，滚动、唤出菜单、放大缩小页面等

三指:上滑打开所属界面、下滑回到桌面布局、上滑打开程序管理与时间轴四指:左右滑动切换桌面等

win10操作系统

桌面设置

更改壁纸:鼠标右键-[个性化]-[背景]-更改壁纸

回收站

回收站:文件删除后的归处，右键点击文件[还原]即可还原回收站清空后数据不可还原

浏览器

桌面图标中包含浏览器，名为IE浏览器(Windows Edge)

图标设置

图标设置:鼠标右键-[个性化]-[主题]-[相关设置]-[桌面图标]

显示设置

屏幕亮度:鼠标右键-[显示设置]-调整亮度(台式电脑则调整显示器亮度)

缩放与布局:更改文本、应用的显示大小，分辨率(越高越清晰)、查看当前显示器属性 声音选项:管理[音频输出]与[音频输入] 蓝牙与其他设备:关闭/开启蓝牙设备，添加打印机与扫描仪等(蓝牙可以链接鼠标、键盘、打印机等支持蓝牙链接的设备)

存储空间:更为详细的观察电脑内部存储空间状态

个性化设置

背景选项:更改电脑壁纸、更改电脑颜色主题

锁屏界面:管理锁屏壁纸与主题

任务栏设置

开始菜单:设置、关机、所有应用程序的菜单页面

网络链接(笔记本可使用Wi-Fi链接，部分台式电脑也可以Wi-Fi连接，大多数台式电脑通过网线连接)

搜索栏:通过输入关键词快速锁定目标应用

任务视图:查看当前任务

中部图标烂:可固定应用程序也可以显示当前已经被打开的程序

输入法

天气:显示天气

时间

隐藏图标:可以随意拖动其中的图标

总结有用的小技巧

关闭弹窗

  1. 打开控制面板
  2. 找到‘Internet选项’并点击。
  3. 在‘Internet选项’窗口中找到隐私列–>点击设置(E)
  4. 在弹出的’弹出窗口阻止程序设置’ 中找到 ‘阻止级别’ 并选择 ‘高’。再点击关闭。

彻底关闭Windows10的自动更新

  1. 选中桌面图标’此电脑’右键管理（G）
  2. 在弹出的‘计算机管理’窗口中打开‘服务和应用程序’一栏中”服务”
  3. 找到‘Windows Update’,右键点击属性。 启动类型设置为禁用、停止服务在弹出的‘Windows Update 的属性’弹窗中切换到「恢复」标签，选择服务失败时的计算机反应无操作。 最后一定要按应用（A）不然无效
  4. 找到【Windows 更新医生服务】，右键属性 启动类型设置为 禁用、停止服务 切换到「恢复」标签，选择服务失败时的计算机反应为 无操作。（提示 拒绝访问 的接着往下看【修改注册表】）
  5. 找到【更新 Orchestrator 服务】，右键属性 启动类型设置为 禁用、停止服务 切换到「恢复」标签，选择服务失败时的计算机反应为 无操作。
  6. 修改注册表win+R 输入regedit(没有拒绝访问这一步忽略)

文件、文件夹和目录等基本概念说明

文件夹:存放文件的容器

文件:存放数据的容器，所有的数据都是以文件形式存储的

目录和目录文件:目录也是文件，是一种特殊文件，叫目录文件，简称目录。

目录文件是全部文件目录组成的文件，用于整个文件系统的管理。

也就是说，目录文件记录着文件所在的具体位置。

术语来说目录指的是文档文件和文件夹的结构化列表存储在计算机上的方式，并且不包含实际文件本身。这就像新华字典前面的目录，存的是字所在页数的位置信息。

这也说明了一个点，就是为什么你1TB的硬盘，其实存不了1TB的数据，因为还有些目录信息数据需要存储，它也是占用空间的。

文件目录:文件目录用于对单个文件的控制，它记录文件的名字、文件长度、文件存放在外存上的物理地址(也叫文件路径)，以及文件属性和文件建立时间、日期等信息。

绝对路径、完整路径，物理路径、真实路径指的都是:这样形式的路径c:too1s\jaden\jaden.txt
